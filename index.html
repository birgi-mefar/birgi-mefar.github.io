# Birgi Mefar IT

# Şirketteki bilgisayarıma evden nasıl ulaşabilrim ?

Şirketteki çalıştığınız bilgisayarın açık olduğundan emin olun.

1. VPN programını indirmek için tıklayınız.

2. Uzaktan erişim yapabilmek için Windows "Uzak Masaüstü Bağlantısı" Programını windows'ta aratıp açın.
Bilgisayar : Bilgisayar Adı
Kullanıcı Adı : DOMAIN\ Core+ Adı QDMS Adı

# Şirket Mail adresime nasıl girebilirim ?

Outlook Web App üzerinden maillerinize erişmek için tıklayınız.

# VPN programını nasıl indirebilirim ?

FortiClient'ı bilgisayarınıza indirmek için tıklayınız.

# VPN kurulumunu nasıl yapabilirim ?

1. Next'e tıkladıktan sonra "install" buttonuna tıklanıldığında kurulum başlıyor.

2. Kurulum bittikten sonra "finish" butonu ile kurulumu bitirin.

3. FortiClient VPN uygulamasını açtığımızda "I acknowledge that this free sof..." kısmını kabul edin.

4. "Configure VPN" tıklandıktan sonra
VPN : SSL_VPN
Connection Name : Mefar VPN
Remote Gateway : 212.57.28.218
Customize Port: 10443
Client Certificate: None
Authentication : Save Login
Username : Bilgisayar Kullanıcı Adı / CorePlus Adı / QDMS Adı

5. "Password" kısmı için bilgisayarınızın şifresini ve ya şifrenizi girin.

6. "Status" kısmı %100 olduğunda "VPN Connected" penceresinde "Disconnected" butonu görünüyorsa firmaya uzaktan bağlı durumundasınız.


# QDMS'e nasıl bağlanabilirim ?

QDMS'i bilgisayarınıza indirmek için tıklayınız.

# Core+'a nasıl bağlanabilirim ?

1. Öncelikle, VPN'e bağlı olduğunuza emin olun.

2. Core+'a şahsi bilgisayarınıza indiremek için tıklayınız.

# Skype'ı bilgisayarıma nasıl kurabilirim ?

1. Skype kullanımı için bilgisayarınızın internete bağlı olmanız ve bir Microsoft accountunuzun olması gerekiyor.

2. https://web.skype.com/ adresine Google chrome veya internete explorer adres barına yazarak veya linke tıklayarak erişilmeli.

3. Skype web sayfasında ekranın sol üst köşesinde bulunan sign in ‘e tıklayarak gelen menüde “use skype online” tıklayarak giriş ekranına ulaşılır.

4. Microsoft hesabınız varsa gelen giriş ekranında email veya telefon numarası ve şifre girilerek skype girişi yapılır. Eğer bir hesabınız yoksa login ekranında “Create one” tıklayarak yeni bir hesap oluşturulur.
